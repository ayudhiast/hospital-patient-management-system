# Hospital Patient Management System

## Project Description

Hospital Patient Management System is a simple Python console application that helps hospital administrators manage patient records.

This project was created as a Capstone Project for Purwadhika.

---

## Features

- Display All Patient Data
- Search Patient by ID
- Search Patient by Diagnosis
- Search Patient by Insurance
- Add New Patient
- Update Patient Data
- Delete Patient Data
- Display Patient Statistics

---

## Dataset Structure

1. Patient ID: Primary Key
2. Patient Name: Patient Name
3. Age: Patient Age
4. Gender: Male/Female
5. Diagnosis: Main Diagnosis
6. Insurance: BPJS/Private/Uninsured
7. Admission Status: Inpatient/Outpatient
